## Structured Notes

### Core Technique
Incorporating real scientific phenomena (meteorology, astronomy, natural patterns) into Houdini VFX — using accurate scientific understanding as the foundation for more believable volumetric clouds, planetary systems, and natural procedural shapes.

### Summary
Kate Zegarace (VFX artist at MPC Toronto, credits: Umbrella Academy, The Boys, Raised by Wolves, The Witcher) teaches a course bridging scientific knowledge with VFX practice in Houdini. Each session introduces the science of a phenomenon before building the corresponding VFX setup. Sessions cover: storm cloud formation and volumetric simulation, procedural planetary systems informed by astronomy, nature's patterns (fractals, Fibonacci patterns) from microscopic to cosmic scales. The approach improves both the technical accuracy and artistic plausibility of VFX work.

### Key Steps
1. Study cloud types (cirrus, cumulus, cumulonimbus) and storm system formation
2. Build volumetric cloud simulations in Houdini based on meteorological structure
3. Learn how solar systems form — accretion, gravitational attraction
4. Build a procedural planetary system with scientifically-inspired orbital mechanics
5. Integrate astronomy data to create a custom star/planet system
6. Study nature's patterns: fractal geometry, Fibonacci spirals, L-systems
7. Build procedural shapes that replicate natural patterns from micro to cosmic scale
8. Apply scientific understanding to improve visual plausibility in FX builds

### Houdini Nodes / VEX / Settings
- Pyro Solver (cloud volumetric simulation)
- Volume VOP (cloud shaping)
- Particle system (planet accretion)
- L-System SOP (natural fractal patterns)
- Procedural modeling (planetary geometry)
- VEX: noise-based natural patterns
- Attribute Wrangle (orbital mechanics)
