## Structured Notes

### Core Technique
Procedural sponge/foam geometry using a box with noise displacement, VDB/SDF operations to create porous cavities, `pointsfromvolume` to scatter interior points with `rest` attribute for coloring, and `pscale` control. Based on Instagram work by Max Shulgi. Russian-language tutorial.

### Summary
A 32-minute Russian-language tutorial by Alexander Eskin recreating a sponge/foam effect inspired by Max Shulgi's Instagram work. Workflow: box (0.7 height, 15 wide) → polygon conversion with higher resolution → default noise displacement → points scattered via `pointsfromvolume` → `rest` attribute added for downstream color/material control → `pscale` attribute for size variation. Final render (frame 003) shows a blue-teal wet sponge aesthetic with porous cavities and subsurface-like material.

### Key Steps
1. Geo node "points_source" → `box` SOP — height 0.7, width ~15; offset upward
2. Convert to polygons with increased resolution for displacement
3. `mountain` or default noise SOP → displace surface to create organic irregularity
4. `pointsfromvolume` SOP — scatter points inside the geometry (preview on top of geo)
5. Add `rest` attribute — required for later material/color-based rendering
6. Clean up: delete unnecessary point groups; keep "posing" group
7. `pscale` attribute → per-point size control; restore via `attribcopy` if needed after copy ops
8. Render with wet/subsurface material (blue-teal sponge aesthetic)

### Houdini Nodes / VEX / Settings
- `box` SOP — height 0.7, width ~15
- Resolution increase — subdivide or resample for noise quality
- `mountain` or noise SOP — default settings for organic surface
- `pointsfromvolume` SOP — interior point scatter (~3800 resolution)
- `rest` attribute — added via `attribcreate` or wrangle; used for coloring
- `attribcopy` or `attribwrangle` — restore `pscale` after copy operations
- Point group cleanup — delete all except "posing"
- Subsurface/wet material in render
