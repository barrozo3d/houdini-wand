## Structured Notes

### Core Technique
Continuation of luxury particle shot: noise/mask polish; File Cache 1M particles (keep only pscale+ID); Voronoi fragment as instance geometry (Box → Scatter 45pts → Voronoi Fracture → Pack → Blast exterior → center); Redshift Instance node; orient randomized from ID; Redshift material (metallic gold); two-shot camera setup (front, top) with per-shot light masks.

### Summary
22m5s tutorial by Alexander Eskin. Part 2 of luxury brand particle shot. Polishes the noise mask (frequency 1.7, Turbulence noise added, ramp with Spline interpolation for sharper contrast peaks). File Cache full simulation (1M particles) keeping only pscale+ID. Creates Voronoi fragment instance geometry: Box → Scatter 45 pts → Voronoi Fracture → Pack (Instancer) → Blast exterior pieces (bounds 0.55) → one fragment → center + normalize scale. Redshift Instance on particle cloud. Orient attribute randomized per ID. Redshift material: metallic gold, soft shadows (softness=8). Two-shot setup: shot_10 (front), shot_20 (top) → separate lights (light_10, light_20) per shot for dramatic rendering.

### Key Steps
1. **Noise polish**: frequency→1.7; Add Turbulence noise to existing mask (Add to mode); ramp (float type) with Spline interpolation → sharper contrast peaks, more visible spikes in mask
2. **File Cache**: naming with `$OS` (node name) + `$F4` (4-digit frame numbers); keep only pscale, ID; delete all other attributes; also delete all groups
3. **Scale full simulation**: increase to ~1 million particles; scale pscale parameter accordingly
4. **Voronoi particle instance geometry**:
   - Box → Scatter 45 points → Voronoi Fracture → Pack (Instancer/Name attribute) → blast exterior pieces (Blast by bounds 0.55) → remain only interior fragments
   - Pick one fragment: unpack → center (Match Size/center) → normalize scale to match preview sphere
   - File Cache the instance geo
5. **Redshift Instance**: create new Instance node → path to cached fragment file → target = particle scatter cloud (1M pts); now each particle shows as Voronoi fragment
6. **Orient randomization** (on points before instancing):
   - Wrangle: `@orient = quaternion(rand(@id) * 2 * M_PI, {0,1,0});` (or random from ID)
   - Each particle rotated differently → organic dust appearance
7. **Camera**: distance 6m; rotate for desired composition; two shots (shot_10 front, shot_20 top)
8. **Light (Redshift)**: area light, distant-type feel; Shadow Softness=8 (softer shadows); adjust brightness
9. **Redshift Material** ("Gold"):
   - Manual specular; metallic; roughness tuned; color: reddish-gold tone
   - Assign to particle instance geo (by path, not geometry in viewport)
10. **Two-shot/light-mask setup**:
    - Light "light_10": for shot_10 (front), specific intensity and direction
    - Light "light_20": for shot_20 (top view), different angle
    - Render target light masks: each shot renders with only its assigned light

### Houdini Nodes / VEX / Settings
- Noise frequency=1.7; Turbulence noise in "Add to" mode; float ramp (Spline) for mask sharpening
- File Cache: `$OS` naming + `$F4`; attribute delete (keep pscale, ID only)
- Voronoi Fracture: Box → Scatter 45pts → Voronoi → Pack → Blast by bounds 0.55
- **Redshift Instance**: path attribute to cached geo file; reads pscale for scale
- Orient wrangle: `@orient = quaternion(rand(@id) * 2 * M_PI, {0,1,0});`
- Redshift material: metallic, manual specular, gold color
- Shadow Softness=8 for soft area shadows
- Render target + light masks: per-shot light assignment (light_10, light_20)
- `$OS` expression: takes file name from node name (convenient for file caches)
