## Structured Notes

### Core Technique
Russian-language Part 2 of Heavy Lux series — refining the particle sim with turbulent noise (frequency 1.7, blended with "Add to" mode), float-type B-spline ramp for mask contrast, `filecache` for sim caching, and final luxury render producing a golden flowing fabric aesthetic. Parallel content to Heavy Chic Part 2 (English).

### Summary
A 22-minute Russian-language continuation (Part 2) by Alexander Eskin completing the "Heavy Lux" particle effect. Covers: noise frequency tuned to 1.7, turbulent noise added via "Add to" blend, float-type (not vector) ramp node with B-spline/Spline interpolation for sharper peak contrast in the mask, sim caching via `filecache`, and final Karma/Mantra render — final frame shows a golden luxury flowing fabric particle aesthetic suitable for brand advertising.

### Key Steps
1. In `attribvop`/`popvop`: adjust noise frequency to **1.7**
2. Add turbulent noise via **"Add to"** blend mode for layered complexity
3. Add ramp node — must be **float type** (not vector, which causes bug)
4. Ramp: B-spline / Spline interpolation, 3 points → sharper peaks, more contrast in mask
5. Apply ramp to noise mask; reconnect to proper output
6. `filecache` SOP — cache sim to disk for playback; set naming convention with `$F` for per-frame files (4 digits: `$F4`)
7. Name the node clearly (e.g., "tex_rus")
8. Render: Karma/Mantra — final result: golden flowing luxury particle cloud

### Houdini Nodes / VEX / Settings
- `attribvop` / `popvop` — turbulent noise, frequency 1.7, blend mode: Add To
- Ramp VOP — **float type** (critical: vector type breaks); Spline interpolation; 3 points
- `filecache` SOP — `$F4` frame naming; caches sim to disk
- Karma/Mantra render node
