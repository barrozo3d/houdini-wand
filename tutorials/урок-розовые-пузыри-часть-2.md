## Structured Notes

### Core Technique
Octane rendering setup for pink bubble scene (continuation of Part 1): Octane Specular (aSpecular) material for glass bubbles; reverse normals for inner bubbles; particle trails added as VFX layer using POP Source on bent-grid tunnel + Trail node + Lines by ID + Octane Strands material with random color per ID and curveu alpha fade.

### Summary
15m49s tutorial by Alexander Eskin. Part 2 of Pink Bubbles project — rendering and scene polish. Exports geometry elements, builds Octane Render Target with 3 shaders (background, outer bubbles, inner bubbles). Background: plane (bottom wall only, SubDiv smooth). Bubble material: Octane Specular (aSpecular mode) → glass look; inner bubbles need Reverse normals for correct refraction. Small/tiny bubbles: Copy to Points or Instance string attribute. Ripple wave: change POP seed so waves originate from different positions; Octane SubDiv for sphere polygon smoothness. Adds particle trails VFX: grid bent 90° as tunnel → POP Source (Z velocity only, life=7, variance=2) → Trail (length 20, Time Shift +500) → Lines by ID → random color + curveu alpha → Octane Strands cylinder material (straight, no subdivision).

### Key Steps
1. **Export elements**: pull out Big Bubbles, Small Bubbles, Tiny Bubbles, Sphere as separate out nodes
2. **Background plane**: keep bottom wall only → Subdivide for smooth normals; set to Roll (not Crease) normals; 6m×3m room
3. **Octane Render Target**: 3 shaders (background, bubbles outer, bubbles inner); Path Tracing mode
4. **Octane Light**: KeyMap light, round type; reposition to -5 Y; reduce size; aim down at ground
5. **Bubble shader** (Octane Specular / aSpecular):
   - Non-standard Specular material type → glass behavior
   - Tint slightly pink
   - Outer bubbles: standard normals
   - Inner bubbles: **Reverse normals** → correct refraction/reflection inside
   - Small bubbles: same shader but paler (lower saturation)
6. **Tiny bubbles**: Copy to Points on sphere OR string instance attribute (faster for 50k points)
7. **Ripple refinement**: change POP seed → waves originate from different surface locations; Ripple Solver attenuation parameter
8. **Octane SubDiv on sphere**: enables for correct reflections on sphere polygon edges
9. **Particle Trails VFX**:
   - Grid → Bend 90° → tunnel/channel shape
   - POP Source inside DOP: 5000–15 particles; velocity Z-only (remove X,Y components)
   - Delete attributes: keep only H (hue), Life, velocity
   - Life=7, Life Variance=2 (short-lived)
   - **Trail node**: points mode, Length=20 (points)
   - Time Shift offset: `$F + 500` → skip to future frames (trails already flowing)
   - **Lines SOP**: bygroup, by attribute "ID" → convert particles to polyline segments
   - **Resample** → curveu attribute (0 at head, 1 at tail)
   - **Wrangle**: `@Cd = random(@primnum);` (random color per ID); `@Alpha = @curveu;` (fade at tail)
   - Fix: `@Cd` must be color type (not float) → change attribute type
   - Octane Material: **Strands material**, Cylinder type; Color=@Cd; Alpha=@Alpha; Intensity=2 (emissive glow)
   - Keep straight (no curve subdivision on strands)
10. **Final**: multiple render target shortcuts (Ctrl+1/2/3) for quick iteration

### Houdini Nodes / VEX / Settings
- **Octane Specular (aSpecular)**: glass shader; apply to outer bubbles; pink tint
- **Reverse SOP**: normals for inner bubble geo → correct refraction
- **Octane SubDiv**: on sphere geo → smooth polygon shading
- POP Source: life=7, life variance=2; delete velocity X,Y → Z only
- **Trail SOP**: points mode, length=20
- Time Shift: `$F + 500` on POP source → trail already flowing at frame 0
- **Lines SOP**: bygroup + by attribute "ID" → particles → polylines
- **Resample** → curveu (for alpha fade on trail)
- Wrangle: `@Cd = random(@primnum);` (per-prim random color); `@Alpha = @curveu;`
- **Octane Strands Material**: cylinder type; straight (no tessellation); Color+Alpha attributes; Intensity=2
