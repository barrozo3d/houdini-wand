## Structured Notes

### Core Technique
Render/material completion of Pink Bubbles series (Part 2): export separate geometry elements from SOP, camera placement, `subdivide` with "roll" (not crease), Octane render target + 3 shaders (Ground, Bubble, Bubble Main), circular light source for glass spheres. Russian-language.

### Summary
A 15-minute Russian-language Part 2 completing the Pink Bubbles effect with Octane rendering. Exports geometry elements separately from the SOP network, sets up camera and scene. Key: `subdivide` uses "roll" parameter (not crease) for the final bubble geometry. Creates Octane render target + 3 Octane shaders (Ground, Bubble, Bubble Main). White/preview materials first; colored in final. Critical: Octane requires at least one light source or render is fully black — adds a circular/disk light (position ~5, 5.6, 5 and -5, -6). Frame 003 confirms white glass spheres rendered.

### Key Steps
1. Export separate elements from SOP geo node (bubbles, ground, outer sphere)
2. Camera placement in scene
3. Leave bottom wall only; keep outer sphere (~6m diameter, inner ~3m)
4. `subdivide` SOP — use **"roll"** parameter (not crease) for smoother geometry
5. Octane render setup: camera → Octane camera; create Octane render target
6. Create 3 Octane shaders: **Ground**, **Bubble**, **Bubble Main** — white/default first for preview
7. Add circular/disk Octane light — without a light source render is completely black
8. Light position: ~5, 5.6, 5 (key) and -5, -6 (fill)
9. Preview → adjust materials; add pink/glass tint in final

### Houdini Nodes / VEX / Settings
- `subdivide` SOP — "roll" parameter (not crease)
- Octane render target
- Octane camera
- Octane shaders: Ground (diffuse/floor), Bubble (glass/SSS), Bubble Main (outer sphere)
- Octane disk/circular light — required; position ~(5, 5.6, 5)
