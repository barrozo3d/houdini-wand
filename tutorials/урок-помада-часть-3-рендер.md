Sets up Octane render node + camera "shot_010", render target "render_target_010" (half resolution, specular depth 4, adaptive sampling), Cryptomatte by material/model name. ### Key Steps
1. Octane camera node — rename to "shot_010", half resolution
2. Octane render target "render_target_010" — specular depth **4**, adaptive sampling, no environment yet → black
3. Camera: tilt focus on, A still mapping
4. Cryptomatte AOV: by material node name (and by model name)
5. Assign render target to context 2; Octane to context 1
6. IPR: confirm black (no lights/environment)
7. Add materials: lipstick body (red wax/SSS), metal cap, plastic tube, water droplets
8. Add lights (area + HDRI) for product lighting
9. Final render: professional red lipstick product shot with droplets
