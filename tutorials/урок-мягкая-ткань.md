## Structured Notes

### Core Technique
Procedural woven fabric geometry built entirely in SOPs using VEX harmonic functions (sine/cosine on `curveu`) — no Vellum simulation. A high-density line curve with 2000 points gets `curveu` from `resample`, then an `attribwrangle` applies sinusoidal displacement to create the interlocking loop pattern of woven cloth. Frequency ~100–200 loops per unit; amplitude and scale promoted as sliders.

### Summary
A 43-minute Russian-language tutorial by Alexander Eskin building procedural woven fabric geometry purely in SOPs via VEX math. A Z-axis line (2000 points, length 1) → `resample` → `curveu` → `attribwrangle` ("weaving") applies `sin(curveu * freq)` to Y and X positions with controllable amplitude and phase offset. Frequency ~100–200 creates fabric-like loops; parameters promoted to sliders (global scale, amplitude X/Y, frequency). The resulting strands are swept/distributed to build a woven cloth surface (frame 002 confirms parallel ridged fabric look). No Vellum required.

### Key Steps
1. Geo node "weaving" → `line` SOP — Z direction, length 1, **2000 points**
2. Color green (`Cd`) for visibility
3. `resample` SOP → generates `curveu` attribute (0→1 along line)
4. `attribwrangle` named "weaving" — apply harmonic displacement:
```vex
// Context: Point Wrangle
float freq = ch("frequency");    // ~100-200 for fabric loops
float ampY = ch("amplitude_y");
float ampX = ch("amplitude_x");
float scale = ch("global_scale");

v@P.y += sin(@curveu * freq * scale) * ampY;
v@P.x += sin(@curveu * freq * scale + ch("phase_x")) * ampX;
```
5. Promote `frequency`, `amplitude_x`, `amplitude_y`, `global_scale` as channel parameters/sliders
6. Set frequency ~100 → 100 loops per unit; 200 for tighter weave
7. Distribute strands in a grid (via `copytopoints` or `for-each`) to build full cloth surface
8. `sweep` SOP for strand thickness
9. Render with Redshift/Karma cloth material

### Houdini Nodes / VEX / Settings
- `line` SOP — Z direction, length 1, 2000 points
- `resample` SOP — generates `curveu` attribute
- `attribwrangle` — `sin(@curveu * freq)` for Y; `sin(@curveu * freq + phase)` for X
- VEX functions: `sin()`, `ch()` for parameter promotion
- Promoted parameters: frequency, amplitude_x, amplitude_y, global_scale, phase_x
- Frequency ~100–200 for realistic fabric loop count
- `copytopoints` or for-each loop — distribute strands
- `sweep` SOP — strand cross-section
