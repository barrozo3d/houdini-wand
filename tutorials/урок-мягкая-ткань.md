## Structured Notes

### Core Technique
Procedural woven/braided fiber animation using VEX harmonic (sine) functions on a line to create interlocking loops (even/odd phase alternation with `@ptnum % 2`); Sweep with twists for 3D geometry; 3-way blend (VEX lerp) between three sweep states (zero-radius, normal, giant-radius) driven by custom `blend` attribute for grow animation without rotation artifacts. Point Deform with relax simulation deforms the weave with cloth-like motion. Hair Shader for thin strand rendering.

### Summary
43m32s tutorial by Alexander Eskin. Creates animated woven/knitted fiber effect. Pipeline: 2000-point line → VEX sine displacement (`Z = sin(curveu * freq) * amp`, `X = sin(curveu * freq - phase) * amp`) → Copy to Points with even/odd phase flip (`@ptnum % 2 ? -1 : 1`) for interlocking loops → Sweep (170 twists, small radius) for 3D tubes → curveu-driven pscale for thickness variation. Grow animation: 3 sweep states (radius=0, normal, giant) + Attribute Blend VEX lerp avoids Carve SOP rotation bug. Point Deform on relax-simulated grid for cloth wave motion. Hair Shader (transparency 35%, Hair globals tessellation 4–8 steps, width=blend attribute). 

### Key Steps
1. **Base line**: Line (Z direction, 2000 points) → curveu via Resample
2. **VEX wrangle ("weaving")** — primary loop displacement:
   ```vex
   float freq = chi("freq");
   float amp = ch("amp_x");
   @P.z = sin(@curveu * freq) * amp;
   @P.x = sin(@curveu * freq + ch("phase_x")) * amp;
   ```
   - Even/odd phase flip: `float sign = (@ptnum % 2 == 0) ? 1 : -1;` → multiply amplitude by sign → loops interlock
   - Export freq, amplitude, phase as slider parameters
3. **Copy to Points**: 2 (or more) points on a line → two fibers at offset X positions
4. **Sweep**: second input = circle profile; small radius; **170+ twists** → tight spiral look; increase for better result
5. **curveu → pscale**: Attribute Wrangle `@pscale = fit01(@curveu, 1, 0);` (tapering tip) → drives Sweep width
6. **Grow animation — 3-state blend** (avoids Carve SOP rotation bug):
   - Sweep state 0: radius=0 (reference/invisible)
   - Sweep state 1: normal petals (final result)
   - Sweep state 2: giant radius (puffed/starting state)
   - Custom attribute `blend` (0→1): `@blend = fit(@Time + 10, 0, 1, 0, 1);` or animated attribute
   - VEX blend wrangle:
     ```vex
     float b0 = fit(@blend, 0, 0.5, 0, 1);  // blend state0→state1 in first half
     float b1 = fit(@blend, 0.5, 1, 0, 1);  // blend state1→state2 in second half
     @P = lerp(lerp(v@rest0, @P, b0), v@bigP, b1);
     ```
   - Place `blend` attribute **after** Sweep (not before) → avoids viewport rotation artifacts
7. **Attribute Noise** on position + AttributeNoise for organic variation
8. **Relax simulation** (Vellum Relax or simple point relax):
   - Vellum Solver + no gravity → gentle relaxing motion
   - Noise inside solver (frequency 0.2, scale 0.3, amp 0.5)
9. **Point Deform**: rest = Time Shift (frame 0 of grid); animated = deformed grid → applies cloth-like wave to weave geometry
10. **Hair Shader** for rendering:
    - Transparency: 35%
    - Width attribute: `@width = @blend * small_value;` (blend drives thickness for grow)
    - Blast points where blend≈0 (removes invisible geometry at start)
    - Hair Globals → **tessellation steps: 4 or 8** (smooths curve angles in render)

### Houdini Nodes / VEX / Settings
- VEX sine displacement: `@P.z = sin(@curveu * freq) * amp;` + phase offset for X
- Even/odd interlock: `float sign = (@ptnum % 2 == 0) ? 1 : -1;` before amplitude
- **Sweep**: profile=circle, radius=small, **twists=170+**; curveu-driven pscale for tapering
- 3-state blend wrangle: `lerp(lerp(state0, state1, b0), state2, b1)` driven by `@blend` attribute
- Vellum Relax Solver: gravity off; noise inside solver for organic motion
- **Point Deform**: rest input (frame 0 grid) + animated input (deformed grid)
- **Hair Shader**: transparency=35%; width attribute=`@blend * scale`; Hair Globals tessellation=4–8
- Blast: remove points where `@blend < 0.001` (invisible geometry cleanup)
- curveu attribute: generated by Resample; used for pscale tapering + blend remap
