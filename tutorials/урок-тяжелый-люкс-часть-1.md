## Structured Notes

### Core Technique
Dense luxury glitter/dust particle effect (POP network) using All Points emission from a grid, wind velocity with VEX mask control via `Cd` attribute, designed for social media luxury brand advertising. Part 1 of a multi-part series. Language: Russian.

### Summary
A 29-minute Russian-language tutorial by Alexander Eskin (Part 1) on creating a "Heavy Lux" luxury particle effect for brand/social media use (final frame shows Dior logo render). Covers POP network setup with All Points emission from 10,000-point grid, stable particle count across frames, wind velocity (value 6) bound to a mask via VEX wrangle using `windgroupraw` (not `windgroup`), and `Cd` attribute for color variation. Frames show: dense glitter particle cloud, glowing oval light source, flat plane geometry, and final Dior brand render.

### Key Steps
1. Create geo node; inside: `grid` SOP → color green via `Cd`
2. `popnet` DOP → inside: `popsource` — Emission Type: **All Points** (all 10,000 grid points become emitters)
3. Set birth parameters so no particle duplication on frame 1 (value ~2.1)
4. Verify stable count: 25 frames → still 10,000 points
5. Add `popwind` or `popforce` → set wind velocity to 6
6. Create curl noise / turbulence mask → store as `Cd` attribute
7. VEX wrangle — bind wind velocity to mask: use `windgroupraw` (NOT `windgroup`, which misleads)
8. Adjust mask for directional particle variation and flow

### Houdini Nodes / VEX / Settings
- `grid` SOP — source geometry; `Cd` attribute set to green
- `popnet` DOP
- `popsource` — Emission Type: All Points; ~10,000 points
- Birth rate: set to avoid duplication on frame 1 (value ~2.1)
- `popwind` — wind velocity: 6
- VEX wrangle — `windgroupraw` parameter for mask-bound wind (not `windgroup`)
- `Cd` attribute — color + mask driver
