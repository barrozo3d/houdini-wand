## Structured Notes

### Core Technique
POP particle rain system: grid emitter with gravity, "Add to inherited velocity" for variation, VDB-based collision with a character geometry using Static Object + Static Solver inside DOP, Pop Stream Group to capture collision hits, rendered in Mantra.

### Summary
33m6s beginner tutorial by Fx Guru (Arbaaj). Builds a rain effect from scratch: a flat grid emitter generates particles in a Pop Network, gravity pulls them down, inherited velocity variation removes per-frame layering artifacts. Collision is set up using a VDB generated by a Collision Source SOP from the character geometry, then fed into a Static Object / Static Solver inside the DOP network. Pop Stream Group captures the "hit" group for particles striking the surface. Note: transcript quality degrades in the final third of the video.

### Key Steps
1. **Scene setup**: place character geometry, scale and position a grid above it as the rain emitter
2. **Pop Network**: create Pop Network → Pop Source (grid as emitter) → Pop Solver → Pop Gravity (enable, toggle on real-time)
3. **Birth settings**: set Life Expectancy ~1; use both Impulse (burst count) and Constant Birth Rate (per-second count); adjust seeds for variation
4. **Velocity variation**: set velocity mode to "Add to inherited velocity" to break up the uniform-layer artifact
5. **Collision setup (SOP side)**:
   - Return to character geometry → Poly Fill → **Collision Source SOP** → generates a VDB collider
   - Set volume offset to "inner" for accurate surface collision; enable "birth between frames" = 0.1 if geometry is animated
6. **DOP collision**:
   - Add **Static Object** and **Static Solver** nodes inside the AutoDOP network → merge into the Pop Solver's inputs
   - Static Object → Geometry tab → Collisions → Proxy Volume = VDB
7. **Pop Stream Group**: add a Pop Stream Group node to tag particles that hit the surface ("hit" group) for further effects (splatter, etc.)
8. **Render**: render in Mantra (convert particles to instances/mesh first)

### Houdini Nodes / VEX / Settings
- **Pop Network** / **Pop Source** — particle emission system; source = grid
- **Pop Solver** — main POP solver
- **Pop Gravity** — applies downward force
- Birth parameters: Impulse (burst), Constant Rate (per second), Life Expectancy, seed
- **Velocity mode**: "Add to inherited velocity" — adds variation to initial velocity
- **Collision Source SOP** — converts geometry to VDB for DOP collision; "inner" volume offset
- **Static Object** (DOP) + **Static Solver** (DOP) → feed VDB collider into sim
- Static Object → Geometry → Collisions → Proxy Volume = VDB
- **Pop Stream Group** — captures collision-hit particles into named group
