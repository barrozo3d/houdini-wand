## Structured Notes

### Core Technique
Pink bubbles inside a large sphere: scattered sphere instances with noise-varied `pscale`, pushed outward along surface normals by their own pscale value (radius) so they sit on the inner surface. Two-stream setup: interior bubbles + surface wave animation. Russian-language Part 1.

### Summary
A 30-minute Russian tutorial (Part 1) by Alexander Eskin building pink bubbles packed inside a large sphere. A polygon sphere (64 cols, scale 2) is split into two streams — one for interior bubble placement, one for surface wave animation. `vdbfrompolygons` → `scatter` (65 points) → `pscale` 0.1 varied by noise multiplication (attribvop) for different bubble sizes. Key: `peak` SOP pushes each bubble along its surface normal by its own pscale (radius), so each bubble sits precisely on the inner surface without clipping.

### Key Steps
1. Geo node "bubbles_source" → `sphere` SOP — polygon, 64 columns, scale 2
2. Split into 2 branches: **Branch A** (inner bubbles) + **Branch B** (surface wave)
3. Branch A: `peak` SOP — offset inward slightly to keep scatter inside sphere bounds
4. `vdbfrompolygons` → VDB SDF of sphere
5. `scatter` SOP — **65 points** (not 1000); add `pscale` attribute = **0.1** base
6. `attribwrangle` "preview" → `i@gl_sphere_points = 1` for viewport sphere preview
7. `attribvop` — multiply `pscale` by noise (type changed from Alligator to smoother) → varied bubble sizes
8. **Key step**: `peak` SOP on scattered points — push along normals by `pscale` → each bubble sits on inner surface at its own radius distance
9. Branch B: surface wave animation (continued in Part 2)

### Houdini Nodes / VEX / Settings
- `sphere` SOP — polygon, 64 cols, scale 2
- `vdbfrompolygons` SOP — SDF conversion
- `scatter` SOP — 65 points; `pscale` = 0.1
- `attribwrangle` — `i@gl_sphere_points = 1` for viewport preview
- `attribvop` — noise × `pscale` for size variation; change noise pattern type
- `peak` SOP — push points along normals by `pscale` (radius); seats bubbles on inner surface
- Key rule: **peak distance = pscale** → bubble surface tangent to sphere wall
