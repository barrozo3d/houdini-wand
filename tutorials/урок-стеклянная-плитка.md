## Structured Notes

### Core Technique
Animated glass tiles rotating procedurally: animated grid moves through space → Attribute Transfer propagates wave "rotor" float attribute onto tile scatter points → VEX quaternion: `orient = qmultiply(orient, quaternion(f@rotor, {0,1,0}));` → tiles flip in a traveling wave. Material blend (3 materials) driven by rotation angle. Bonus: particle trails VFX (POP Source → Trail → Lines by ID → Octane Strands).

### Summary
32m14s tutorial by Alexander Eskin. Creates animated glass tile wall with procedural wave flip rotation. Pipeline: tile box (0.1×0.5) → PolyBevel → Mountain → SubDiv; scatter on 128×128 plane grid → Copy to Points with pscale; animated wave attribute via Attribute Transfer from moving grid (dense 55×55, falloff/blend); noise on attribute for organic variation; VEX quaternion for orient; 3 Octane materials blend per rotation state (glass dark, glass light emitting, plain glass). Extra scene: particle trails in bent tunnel (POP → Trail → Lines by ID → random color + curveu alpha → Octane Strands).

### Key Steps
1. **Tile geometry**: Box (0.1×0.5×0.1) → PolyBevel (crease) → SubDiv → smooth glass-like tile
2. **Tile placement**: Grid plane (Y-Z axis, 10×5, height 2.5) → 128×128 points → Copy to Points; Adjacency mode; pscale=visual fit
3. **Wave attribute ("rotor")**:
   - Create dense grid (55×55) — same size as tile plane
   - Animate translation (moves in time) → creates traveling wave shape
   - **Attribute Transfer**: source=animated moving grid; attribute="rotor" (default=0, max=1 white)
   - Falloff/blend parameter on transfer → controls wave edge softness
   - Add Noise to attribute (X-noise only removed) → organic variation
   - **Attribute Copy** back onto tile scatter points
4. **VEX rotation** (Wrangle on tile points):
   ```vex
   orient = qmultiply(orient, quaternion(f@rotor * M_PI, {0,1,0}));
   ```
   - Tiles flip 180° when rotor=1; partially rotated in-between
   - Change axis ({0,1,0} vs {0,0,1}) for different flip direction
5. **Camera + render target**: 196 frames; Octane Camera; 3 render targets
6. **Octane Materials** (3-way blend via rotor/rotation):
   - Material 1: dark glass (base tiles)
   - Material 2: emissive light glass (front-facing when rotated to camera)
   - Material 3: plain glass (back-facing)
   - Material blend: Octane Mix Material driven by rotation angle (blend coverage)
7. **Trails VFX** (added to scene):
   - Grid → Bend 90° (90° angle) → tunnel shape on floor/background
   - POP Source: 5000–15 particles; velocity Z only
   - Delete attributes; Life=7, Life Variance=2
   - **Trail node**: 20 points length; Time Shift +500 (mid-future state)
   - **Lines SOP** (by ID attribute) → polylines per particle
   - **Resample** → curveu attribute
   - Wrangle: `@Cd = random(@primnum);` color per line; `@Alpha = @curveu;` tail fade
   - **Octane Strands Material**: Cylinder type; straight; Color+Alpha+Intensity=2 (glowing)
   - Assign: "Trails" → Strands material

### Houdini Nodes / VEX / Settings
- **Box**: size 0.1×0.5×0.1 (thin flat tile)
- **PolyBevel** + **SubDiv**: crease edges for glass look
- **Mountain SOP**: slight surface irregularity
- Grid 128×128 → **Copy to Points** + Adjacency (pscale for spacing)
- Dense grid 55×55 + animated Transform → **Attribute Transfer** ("rotor") with falloff blend
- Attribute Noise: X component only (removes X noise, keeps YZ)
- **Attribute Copy** SOP: copy "rotor" from dense grid back to tile grid points
- VEX: `orient = qmultiply(orient, quaternion(f@rotor * M_PI, {0,1,0}));`
- Quaternion axis: `{0,1,0}` for Y-axis flip; `{0,0,1}` for Z-axis
- Octane Material Builder: 3 materials (dark, emissive, plain glass); Mix Material for blend
- POP Source → Trail (20 pts) → Time Shift `$F+500` → Lines by ID → Resample → curveu
- Wrangle: `@Cd = random(@primnum);` + `@Alpha = @curveu;`
- Octane Strands: Cylinder type; no curve sub; Color, Alpha, Intensity attributes
