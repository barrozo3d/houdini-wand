## Structured Notes

### Core Technique
Russian-language extended version of the Glass Tiles tutorial — same `box`/`polyextrude`/`subdivide`/`copytopoints` base workflow, but adds **per-tile rotation animation** via `attribtransfer` driving an `orient` quaternion attribute. Final frame shows dynamic glowing light-trail tile animation.

### Summary
A 32-minute Russian-language tutorial (7 minutes longer than the English counterpart) by Alexander Eskin covering the full glass tile pipeline plus tile rotation animation. Builds a single tile (box 0.1×0.5, crease/subdivide bevel), distributes on 128×128 point grid (YZ, 10×2.5), scales via `pscale`. The extended section creates a large secondary geometry to generate a noise-based attribute via `attribtransfer`, which drives per-tile `orient` quaternion rotations — producing an animated wave of rotating glass tiles. Final frame shows glowing light trail aesthetic.

### Key Steps
1. Geo node "Tiles" → `box` SOP (0.1 × 0.5)
2. Color primitive 5 red → group for orientation reference
3. `polyextrude` primitives 4+5 (offset 0.5 / inset); `subdivide` — crease mode, 2 divisions
4. `grid` SOP — YZ plane, size 10 × 2.5, 128×128 points only
5. `copytopoints` SOP; `pscale` → scale tiles to fit (visually adjust)
6. Create large secondary geometry (grid/sphere) for attribute source
7. `attribtransfer` SOP — transfer noise/curvature attribute from large geo to tile points → drives `orient`
8. `orient` quaternion attribute on points → each tile rotates based on transferred value
9. Animate the source attribute over time → tiles rotate in waves
10. Redshift glass shader + refraction render
