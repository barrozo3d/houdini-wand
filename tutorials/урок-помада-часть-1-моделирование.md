## Structured Notes

### Core Technique
Hard-surface lipstick modeling in Houdini: curve → Revolve for body; Box Boolean cut for angled tip; VDB from alpha-texture image for engraved logo cutouts with VDB Sample for curved extrusion; PolyExtrude logo; Cylinder cap with PolyBevel/Stitch for clean edges; SubDiv; Octane materials.

### Summary
15m37s tutorial by Alexander Eskin. Models a lipstick product for render. Part 1 of a series (Parts 2/3 cover FLIP simulation and rendering). Workflow: draw profile curve → Revolve (8–19 divisions) → Fuse pole → Boolean cut angled tip with Box (5–6° tilt) → SubDiv; Alpha texture image → VDB for engraved logo cutouts → Hole node → Divide (convex off, crack thickness 0.3) → VDB Sample 0.1 for curved surface bending → Extrude logo by surface normal; Cylinder tube/cap with PolyExtrude, Stitch (distance 0.2), normal angle limit for no smoothing on edges, PolyBevel; UVs (UV Project) needed for Octane texture coordinates. Materials: lipstick, metal, cap, background. Background: simple plane.

### Key Steps
1. **Profile curve**: draw lipstick profile shape with Pen tool (approx. 6–8 points)
2. **Revolve**: 8–19 divisions → creates lipstick body revolution
3. **Fuse**: small value at pole → cleans up top vertex
4. **Angled tip cut**: Box (longer, Transform 5–6° tilt) → Boolean (subtract) → cuts angled top
5. **SubDiv**: ~5 level for smooth result
6. **Engraved logo cutouts**:
   - Alpha texture image (from project textures folder) → VDB From Polygons → VDB SDF → Hole node
   - Divide node: convex=OFF, Crack Thickness=0.3 (avoids bugs at value 3 or 30)
   - VDB Sample value=0.1 → bends/conforms logo to curved surface so extrusion follows surface
7. **Extrude logo**: Ray SOP to project logo → PolyExtrude by surface normal (minus Z, small amount)
8. **Clean artifacts**: Clean node, rebuild groups after operations
9. **Tube cap (cylinder)**:
   - Cylinder, radius/scale adjusted
   - PolyExtrude ×4 for thicker wall
   - Stitch: divisions 2, distance 0.2 for seam
   - Normal angle: change to prevent smoothing on cylinder hard edges
   - PolyBevel for edge finishing
   - Fuse duplicated vertices
   - Same SubDiv divisions and exclusions as lipstick body
10. **UVs**: UV Project → needed for Octane texture coordinate (flake particles for reflective parts)
11. **Materials**: lipstick, metal, cap/moda, background (Octane Toon Material Builder: 4 materials)
12. **Camera**: aspect ratio set; shot composition

### Houdini Nodes / VEX / Settings
- **Revolve**: 8–19 divisions (not critical)
- **Boolean** (Fuse + Boolean subtract): Box with Transform tilt 5–6° for angled cut
- **VDB From Polygons** → **Hole** node: from alpha texture → engrave effect
- **Divide**: convex=OFF, Crack Thickness=0.3 (bug-free value)
- **VDB Sample**: value=0.1 for surface conforming before logo extrusion
- **PolyExtrude**: along surface normal for logo embossing; ×4 for cylinder wall
- **Stitch**: divisions=2, distance=0.2 for seamless cylinder
- **Normal angle edit**: prevent smoothing on hard cylinder edges
- **UV Project**: for Octane texture coordinates (sparkle/flake textures)
- **SubDiv**: level 5; Octane renderer (progressive, not bucket — faster for this scene)
