## Structured Notes

### Core Technique
Animated glass donut using torus + `aanoise` flow noise with seamlessly looping animation: the `flow` parameter of anti-aliased flow noise cycles 0→1 over 300 frames creating a perfect loop. Critical: animate to frame **299** (not 300) since frame 300 is identical to frame 0. `polyextrude` for thickness, Redshift glass material.

### Summary
A 13-minute Russian-language tutorial building an animated glass donut with flowing surface noise. Uses a `torus` primitive (X-axis, radius 1.5, top/bottom segs 1), applies `aanoise` (anti-aliased flow noise) — the `flow` parameter is animated 0→1 linearly across 300 frames for a seamless loop. Key gotcha: set end frame at 299 (not 300) to avoid duplicate frame at loop boundary. `polyextrude` adds thickness 0.05. Final render (frame 003) shows green/teal glowing torus with flowing animated lines on the surface.

### Key Steps
1. Geo node "glass_donut" → `torus` SOP — X-axis, radius 1.5, top/bottom segments 1
2. `attribvop` or `mountain` with `aanoise` — apply anti-aliased flow noise to surface
3. Noise settings: VEX position multiplied by circle UV (u=0,0,25; v=0,0,25), octaves 4; promote `flow` parameter
4. **Seamless loop trick**: animate `flow` parameter linearly from 0 (frame 1) to 1 (frame **299**) — NOT frame 300, which would duplicate frame 1 and cause a visible stutter at loop point
5. `polyextrude` SOP — thickness **0.05**; output back-faces
6. Camera setup; HDRI lighting
7. Redshift glass/refraction material with green/teal tint

### Houdini Nodes / VEX / Settings
- `torus` SOP — X-axis, radius 1.5, top/bottom segs 1
- `attribvop` — `aanoise` (anti-aliased flow noise); octaves 4
- `flow` parameter: animate **0 → 1 over frames 1–299** (linear); NOT frame 300
- `polyextrude` SOP — thickness 0.05
- Redshift glass shader — refraction + tint
- Key rule: **loop = 299 frames, not 300** (frame 300 = frame 0 = duplicate)
