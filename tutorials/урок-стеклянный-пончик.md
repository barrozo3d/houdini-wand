## Structured Notes

### Core Technique
Glass donut with animated seamless noise loop: Torus + Mountain noise → animated offset 0→1 linear (noise at X=0 identical to X=1, enabling perfect seamless cycle) → PolyExtrude for thickness; gradient textures created in COPs as HDRI dome light; Redshift glass shader (Reflection=1, Roughness=0, Transmission=1); Redshift SubDiv displacement for artifact correction.

### Summary
13m15s tutorial by Alexander Eskin. Quick product-style glass donut render. Creates Torus primitive with Mountain noise displacement → key insight: noise values at offset 0 and 1 are identical → animate offset 0→1 over 300 frames (linear) → set frame 300 keyframe equal to frame 1 value → perfect seamless loop. PolyExtrude for glass thickness. Gradient textures built in Houdini COPs (4000×2000 circle gradient, two variants) → used as dome light textures in Redshift. Standard Redshift glass material. Fix jagged edges: Redshift Object Properties → Displacement → SubDiv max=2 (more causes artifacts with noise).

### Key Steps
1. **Torus**: x-axis radius ~1.5, cross-section 0.6; rotate 90°; few initial divisions
2. **Mountain SOP noise**: default → key property: noise at offset 0 and 1 are identical (perfect loop)
3. **Seamless animation**: animate noise offset 0→1 from frame 1→300 (linear); copy frame 300 keyframe = frame 1 value; loop 299 frames (not 300 to avoid duplicate)
4. **PolyExtrude**: thickness=0.05; output geometry for glass walls
5. **Camera**: FOV 120; lock shot
6. **COPs gradient textures** (two variants):
   - New COP network: width 4000, height 2000
   - Circle SOP → position 0,45 for off-center fade; scale soft edge → gradient shape
   - ColorVibrancio plugin (free) or default Tint for coloring
   - Save two variations (stretched/different orientation) to project/gradient folder
7. **Redshift render** (ShiftCast):
   - Camera selection; disable Global Illumination if too slow
   - Standard Redshift material: Reflection=1, Roughness=0, Transmission=1 → glass look
8. **Dome lights**: two Redshift DnLights; load gradient textures; color space=sRGB; rotate/position
9. **SubDiv for artifacts** (Redshift Object Properties → Displacement):
   - Enable Displacement → Enable SubDiv
   - Min Edge Length=0; Screen Space Adaptive=OFF; Max Subdivisions=2
   - Warning: too many SubDiv with dense noise = unfixable artifacts

### Houdini Nodes / VEX / Settings
- **Torus**: x-axis radius 1.5, cross section 0.6; Mountain SOP noise (default)
- **Noise seamless loop**: offset animated 0→1 over frames 1→300; linear; frame 300 key = frame 1 value
- **PolyExtrude**: thickness=0.05; glass wall output
- COP network: Circle, Tint/ColorVibrancio, export EXR/PNG gradient
- Redshift Standard Material: Reflection=1, Roughness=0, Transmission=1
- Redshift Dome Light (DnLight): texture input; color space=sRGB
- Redshift Object Properties → Displacement: Enable; SubDiv max=2; Screen Space Adaptive OFF
