# ingest.py (Step 1) doesn't call ffmpeg, but checks for it on PATH — if missing,
# it marks the tutorial frame_status: skipped (permanently no frames) instead of
# pending-selection.

# Windows: winget install deno
# Required by yt-dlp to solve YouTube's n-challenge (format/download throttle bypass).
# Without deno: yt-dlp may fail or return incomplete video format data.
